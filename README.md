## OmegaTeam®

OmegaTeam® è un software per Lego Mindstorms EV3 progettato in occasione della RoboCup 2016. Il programma comprende LineFollower e Rescue.

#Termini di utilizzo
Si avvisa la gentile utenza che OmegaTeam® è protetto sotto la lincenza APACHE 2.0, per ulteriori informazioni consultare il documento "License.txt".
Sarebbe gradita una segnalazione ai creatori per un eventuale riutilizzo del programma, o singole parti di esso.
